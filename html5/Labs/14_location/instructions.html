Mapping

Overview

The goal of this lab is to display a Google map centered on your current location (using geolocation). The page.html file in the before folder contains all the libraries and markup you need (the Google map APIs are already included, and as of V3 you no longer need an API key to use the API).

All you'll need to do is add some script to the page to display the map correctly. You might find the following code helpful in centering a Google map on a particular position. Make sure you provide an error handler for the geolocation call. Try to display a message after turning off geolocation to see how the error handling works. 

        
var position = new google.maps.LatLng(lat, lng);
var options = {
     zoom: 8,
     center: position,
     mapTypeId: google.maps.MapTypeId.ROADMAP
};
var map = new google.maps.Map(document.getElementById("map_canvas"), options);
