JavaScript Koans

For this lab we'll be using the JavaScript Koans created by Liam Mclennan. If you haven't heard about programming koans, see: Learning Programming Languages with Koans

The goal of this lab is to open up the page.html file in the before directory of this lab and make all of the QUnit tests pass. You'll do this not by editing page.html (it is just a test runner), but by editing the .js files in the before\topics folder.

Some of the koans are easy, some of them you might find quite challenging. Feel free to search the internet for reference material and understand the true meaning behind a test that stumps you. If you are blocked by a test, try using a debugger, or ask the instructor for help. In the worst case scenario, refer to the after directory for the solution.
