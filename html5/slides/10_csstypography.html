@font-face

  • No longer stuck with "web-safe" fonts
  • @font-face allows browser to download a font
This is some text!
            
@font-face {
    font-family: DeliciousRoman;
    src: url(http://www.font-face.com/fonts/delicious/Delicious-Roman.otf);
    font-weight:400;
}

.delicious {
    font-family:DeliciousRoman;
}
