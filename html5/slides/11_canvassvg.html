Drawing with Canvas and SVG

The Canvas

  • A scriptable bitmap for 2d graphics
  • Simple API - 45 methods
  • Starts out blank
  • Extreme
        
<canvas width="300" height="200"></canvas>

Canvas Context

  • Gateway to the API
  • Doesn't remember what you do...
  • 0,0 is upper left
        
var canvas = $("#myCanvas").get(0);
var context = canvas.getContext("2d");
context.fillRect(25, 25, 250, 150);

Drawing Boxes

  • fillStyle, fillRect
  • Doesn't remember what you do...
  • To clear a canvas, set a dimension property
        
ctx.fillStyle = "rgba(255, 0, 0, 0.5)";
ctx.fillRect(25, 25, 250, 150);
ctx.fillStyle = "#00ff00";
ctx.fillRect(0, 0, 50, 50);
ctx.clearRect(170, 90, 20, 20);

Dynamic Canvas

        
for (var i = 0; i < 6; i++) {
    for (var j = 0; j < 6; j++) {
        ctx.fillStyle = 'rgb(' + Math.floor(255 - 42.5 * i) + ',' +
                         Math.floor(255 - 42.5 * j) + ',0)';
         ctx.fillRect(j * 30, i * 30, 30, 30);
    }
}

Paths

  • moveTo
  • lineTo
  • stroke
        
for (var x = 0; x <= 300; x += 10) {
    ctx.moveTo(x, 0);
    ctx.lineTo(x, 200);
}
ctx.stroke();

Arcs

        
var canvas = $("#canvasArc").get(0);
var ctx = canvas.getContext("2d");

ctx.arc(150, 100, 50, 0, 2 * Math.PI);
ctx.fill();

Arcs and Curves

  • arc
  • quadraticCurve
  • bezier curve

Text

  • font
  • fillText

ctx.font = "45px sans-serif";
ctx.fillStyle = "#ff0000";
ctx.fillText("Hello!", 75, 100);

Gradients

  • Linear gradients
  • Radial gradients

var radgrad = ctx.createRadialGradient(0, 150, 50, 0, 140, 90);
radgrad.addColorStop(0, '#F4F201');
radgrad.addColorStop(0.8, '#E4C700');
radgrad.addColorStop(1, 'rgba(228,199,0,0)');
ctx.fillStyle = radgrad;
ctx.fillRect(0, 0, 300, 200);

Saving and Restoring

  • Settings are global
  • Use save and restore
        
ctx.save();
ctx.rotate(5 * Math.PI / 180);
ctx.fillStyle = "#00ff00";
ctx.fillRect(150, 100, 45, 45);
ctx.restore();

Canvas Security

  • Protection against information leakage
  • origin-clean flag
  • Check on getImageData()
  • Origin checks on toDataUrl()

Canvas Accessibility

  • Fallback content
  • Focus management (very new)

SVG

  • Scalable Vector Graphics
  • Geometry based
  • Full DOM support
  • New York

Basic SVG

        
<svg width="200" height="200">
    <circle cx="100" cy="100" r="50" fill="red">                    
    </circle>
</svg>

SVG and the DOM


.svgcircle
{
    fill:red;
    stroke-width:5;
    stroke:black;
}

SVG and Script


$(".svgcircle").click(function () {
    var circle = $(this);
    circle.attr("r", parseInt(circle.attr("r")) + 10);
});

Canvas versus SVG

Feature Canvas SVG
Abstraction Pixels Shapes
Interaction Script Script or CSS
Perf Small surface, large # objects Large surface, small # objects
Capture Video and Images Static

Summary

  • SVG for detail and scalability
  • Canvas for low level pixel work
