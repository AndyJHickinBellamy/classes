CSS 3 Animations and Transitions

Transitions

  • Properties change over time
  • Can I Use?
This <div> will transition 

.easy {        
    transition-property: background;
    transition-duration: 2s;
    transition-timing-function: ease;        
}

Timing

  • ease
  • linear
  • ease-in
  • ease-out
  • ease-in-out
  • cubic-bezier (ceasar)

Transition Delay

  • transition-delay to delay the transition
        
.easy {        
    transition-property: background;
    transition-duration: 2s;
    transition-timing-function: ease;        
    transition-delay: 0.5s;
}

Transition States

  • Query property during transition will show intermediate state
  • Automatic reversals if interrupted
This <div> will transition 

.easy {        
    /* shortcut syntax */
    transition: background 2s ease;
}

  • Colors, gradients
  • Sizes, positions

Translation and Flow

  • Similar to position:relative
  • Translations don't impact the normal flow
  • Always translate with respect to self
  • Some browsers only allow translation of block elements
        
.scalerotate
{
    -webkit-transform: scale(2) rotate(15deg);
}
