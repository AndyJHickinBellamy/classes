Lab 04 - CSS Fundamentals (and a contest!)

The Contest

The Microsoft MIX 2008 conference held a "remix" contest to redesign the MIX web site using CSS (in the spirit of CSS Zen Garden).

In this lab you can redesign the "MIX home page" which you'll find in the before directory of this lab. Don't modify the HTML source (except to add a stylesheet link). You have 60 minutes to complete a design and upload to the instructor's shared drive.

Check out the after directory for one redesign attempt. Surely, you can do better than that!

Here are a couple ideas:

  • Turn the links in the navigation section into a menu with hover effects specified in CSS.
  • Find a pleasant looking background image.
  • Use rounded corners and shadows.
  • Experiment with font-size settings
  • Use margins to allow space between elements.
