HTML 5 Media

Properties, Events, and Methods

The goal of this lab is to make you comfortable with the properties, methods, and the events of the HTML 5 video attribute. Refer to those previous links for the full reference.

Open up the page.html file in the before directory of this lab. Your goal is to apply what you've learned about JavaScript and what you've learned about HTML 5 video and add the following to the page.

      - A video that will play in your browser. You can search the web for downloadable clips.
      - Hook up all the buttons on the page to interact with the video (play, pause, etc).
      - Wire up all the events in the events table to display the total number of times the events have fired.
      - Every 1 second, refresh the property values in the property table.

If you are clever, you can do all of the above with less than 50 lines of JavaScript code. It is not nessecary to use a JavaScript library, but you can use any framework or library that you already know. Refer to the page in the after folder if you want some additional hints.
