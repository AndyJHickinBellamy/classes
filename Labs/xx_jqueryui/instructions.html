Lab XX - jQuery UI

Overview

In this lab we'll download jQuery UI and use the features inside on the jqueryui.html page in the before folder of this lab.

Getting Started

  1. Open the jqueryui.html page from the before directory of this lab in a browser.
  2. Note the page is not using any jQuery UI components.
  3. Go to jqueryui.com and download jQuery UI. Make sure to select all components. Make sure to pick your favorite theme.
  4. Extract the download into the before directory of this lab.
  5. Your page will need to include a script reference to jQuery. You'll find a minified version in the js folder of the extracted download.
  6. Your page will also need to reference jQuery UI (also in the js folder of the download).
  7. Add a link to the style sheet for jQuery UI. You'll find it in the css directory.
  8. Create a script.js file in the before folder. Add a script tag to reference this file from the jqueryui.html page. The script.js file is where we will write all of our script.

At this point your header should look like the following.

			
<head>
    <title>jquery ui</title>
    <link rel="stylesheet" href="css/vader/jquery-ui-1.8.16.custom.css" media="screen">
    <script src="js/jquery-1.6.4.js"></script>
    <script src="js/scripts/jquery-ui-1.8.16.custom.js"></script>
    <script src="script.js"></script>
</head>

Accordian

In the script.js file you created, add the code to fulfill the following requirements for the accordion section of jqueryui.html. Use the jQuery UI documentation to help complete the requirements.

  1. The "accordion" div should turn into an accordion widget with two headers
  2. The icons on the headers should be triangles inside circles ("ui-icon-circle-triangle e and s")
  3. Allow all sections of the accordion to collapse
  4. The accordion should initially appear with the second section open

Refer to the script in the after folder if you get stuck.

Autocomplete

Add code to script.js to complete the following requirements

  1. When the user types in the input with an id of technology, setup autocomplete functionality with the following technologies: Windows, Windows Phone, Office, Azure, ASP.NET.
  2. The user should type at least 2 characters before the autocomplete list appears

Button

Add code to script.js to complete the following requirements

  1. Turn the button, link, and submit button in the buttons section into jQuery themed buttons.
  2. Turn the radio buttons into a jQuery UI button set.
  3. Turn the checkboxes into a jQuery UI button set.

Datepicker

Add code to script.js to complete the following requirements

  1. Add a datepicker to the input with an id of "theDate".
  2. Don't allow the datepicker to move past January 1, 2011
  3. Allow the user to change the year by selecting the year from a drop down list (hint: see changeYear).

Dialog

Add code to script.js to complete the following.

  1. Create a dialog from the div with an id of "dialog".
  2. Don't let the dialog automatically open when the page loads.'
  3. Add a button in the dialog section that, when clicked, opens the dialog.
  4. Add an "Ok" button to the dialog. When the user clicks the button the dialog should close.

Progressbar

In the progressbar section, see if you can create a progressbar that moves from 0 to 100 over 20 seconds. Hint: use setInterval.

Slider

In the slider section, create a slider widget. Add a div to the section that will display the value of the slider when it changes.

Tabs

In the tabs section, move all the content into tabs that will appear when the user hovers the mouse over the tab header.

Draggables

In the draggables section, make the "Draggables" header draggable, but only left and right.

Droppable

In the droppable section, allow the "Droppable" header to drop into the red and blue blocks. When the header is dropped on these blocks, change the color to white. Make the header revert to its previous position if not dropped on a valid drop target.

Complete!

You've completed the jQuery UI lab and should have a solid understanding of this wonderful library!
