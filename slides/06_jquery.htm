Events

  • Dedicated events methods (click)
  • Generic bind method
        
$("a#openMenu").click(function(){
    $("#menu").show();
    return false;
});

Delegted events

  • Effecient with large # of elements
  • Catch events from future elements
        
$("#dataTable tbody tr").on("click", function(event){
	alert($(this).text());
});

$("#dataTable tbody").on("click", "tr", function(event){
	alert($(this).text());
});

Animations

  • Sliding
  • Fading
  • Custom animations
        
$("#slideTheCode").click(function () {
    var target = $("slideTheCode").next("pre");
    if (target.is(":visible")) {
        target.slideUp();
    } else {
        target.slideDown();
    }
});
