Themes

  • Download an existing theme
  • Roll your own themes
  • Just start from scratch (ugh!)

The jQuery UI Way

  • Every method takes an options object
  • Every widget has events and methods
  • Every widget expects a default HTML structure
$(function(){
	
    $("#myelement").accordion({collapsible:true});

});

Options

  • Options always documented
  • Options always have a default
  • You can also find options in the source
$(function(){

    $("#accordion").accordion({collapsible:true,autoHeight:false});

});

Icons

  • Icons are CSS sprites
  • Icons placed using CSS
$(function(){

    $("#accordion").accordion({collapsible:true,autoheight:false,
                              icons: { header: "ui-icon-arrow-1-s", 
                                       headerSelected: "ui-icon-arrow-1-e"}
    });

});

Methods

  • Not invoked like you might think...
  • Common methods inherited
    • enable
    • disable
    • option
			
$("#accordion").accordion("activate", 1);

Events

  • Every event has event and ui parameters
  • ui properties depend on widget and event
		
$("#accordion").accordion({change: function(event, ui){
	$("body").append($("div").text(ui.newHeader));
}});

                               select: function() { $("#form").submit() },

Datepicker

  • Adds a calendar widget to an input control
  • Keyboard accessible
  • Localizable

Datepicker Localization

  • Use a localized script from jqueryui.com
  • Script will set default locale
  • Can also set locale programmatically
             
$( "#datepicker" ).datepicker( $.datepicker.regional[ "fr" ] );

Other Datepicker options

  • maxDate
  • minDate
  • numberOfMonths
  • showButtonPanel
  • showAnim
 
$(function(){
    $("#theDate").datepicker({showWeek: true, numberOfMonths:2}); 
});

Datepicker events

  • onSelect
  • beforeShowDay ([selectable, class, tooltip])
$(function(){
   $("#theDate").datepicker({
       beforeShowDay: function(date){
           if(date.getDate() == 11){
               return [false, "*", "not the 11th!"]
           }
           return [true, "*", ""];
       }
   }); 
});

Datepicker methods

  • hide
  • show
  • getDate
  • setDate

$("#theDate").datepicker("setDate", "1/1/2001");

Draggables

  • Doesn't work with touch by default

    $("#draggable").draggable();
});

  • grid
  • start
  • drag
  • stop
