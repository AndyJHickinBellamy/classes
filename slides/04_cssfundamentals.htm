What is a selector?

  • Pattern used to identify elements
body {
  background-color: #cccc99;
}
#menu {
  background-color: #ff0000;
}
.tookTitle {
  font-style: "italic";
}

Other selectors

/* descendant */
div p {
	background-color: #ddddaa;
}
/* child */
div > p {
	background-color: #ddddaa;
}
/* attribute */
img[alt=spacer] {
	padding: 0px;
}
/* psuedo class */
a:visited {
	color: #dddddd;
}

How to specify values

  • Keywords
    • thin, thick, larger
  • Physical measurements
    • in, mm, pt, pc
  • Relative measurements
    • %, em
  • Colors
    • #rrggbb, rbg(r,g,b)
  • Fonts
    • Helvetica, sans-serif

Colorful CSS3

  • RGBA adds an alpha component
  • No official support for #rrggbbaa
  • There is a specification for rgba(r,g,b,a)
<div style="background-color:rgba(255, 0, 0, .2)"></div>

HSL

  • Hue (from red to magenta)
  • Saturation (% from gray to full hue)
  • Lightness (% black to white)
  • More intuitive for designers
  • Use hsl, or hsla
<div style="background-color:hsla(200, 50%, 50%, .2)"></div>

Opacity

  • Official in CSS 3
  • Sets opacity for an element and it's children.
<div style="opacity:0.2">
	<div style="background-color:#ff0000"></div>
	<div style="background-color:#00ff00"></div>
</div>

The Cascade Hasn't Changed

  • User agent style sheet (least important)
  • User stylesheet (more important)
  • Author style sheet (most important)
  • Inline styles (always win)
  • Last rule wins

CSS Resets

  • A popular feature in many CSS frameworks
  • The original Meyer reset.
  • Also in Bootstrap, 960gs, and many more
html, body, div, span, applet, object, iframe ... {
	margin: 0;
	padding: 0;
	border: 0;
	font-size: 100%;
	font: inherit;
	vertical-align: baseline;
}

Specificty Is The Same

  • #output trumps div

Inheritance

  • The DOM puts elements into a parent-child relationship
  • Children inherit some property values from parents
    • font-size: yes
    • border: no
